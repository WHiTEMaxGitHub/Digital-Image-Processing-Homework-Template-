### 运行方法：

编译需要opencv库支持，使用cmake构建项目，运行请加参数<i>（含义为第i次作业）与输入文件夹路径。

例：

```bash
./<可执行程序名> 1 <图片文件夹路径>
P.S. 作业2的两个内容分别为
```

输入文件夹在可执行文件的**同级目录下的data/作业名/input**，输出会在对应输入文件夹的**同级目录**建立```output```目录，命名规则如下：

```bash
作业1：
"原文件名" + "_processed" + "_${插值方法缩写}" + "_${缩放倍数}" + "文件扩展名"
```

可执行文件本身在```项目根目录/out/bin```中，使用Linux可以直接运行，windows下请先使用cmake构建项目再编译运行，如果希望使用Linux再次构建与编译项目，可以使用**我提供的Makefile文件**，按照以下方式执行：

```bash
make <debug/release> #设置了两种项目构建模式
make build
# 或者可以直接使用如下方式完成构建 + 编译
make <debug-build/release-build>
```

其会在项目根目录创建**compile_commands.json与可执行文件的软链接**，可以直接在项目根目录运行。
